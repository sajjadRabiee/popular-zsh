# Command Reference

## `padd`

Save a command:

```zsh
padd gs git status
```

## `p`

Run a saved command:

```zsh
p gs
```

Run a templated command:

```zsh
p serve --port=8000
```

## `pls`

Show a polished list of saved commands with template option hints.

```zsh
pls
```

## `premove`

Delete a saved command:

```zsh
premove gs
```

## `pedit`

Open the command file in your editor:

```zsh
pedit
```

## `phelp`

Show the built-in help:

```zsh
phelp
```
