# Examples

## Git shortcuts

```zsh
padd gs git status
padd gp git pull
p gs
```

## Local server

```zsh
padd serve 'python3 -m http.server {{port}}'
p serve --port=3000
```

## Docker

```zsh
padd up 'docker compose up -d'
padd down 'docker compose down'
p up
```

## Project commands

```zsh
padd test 'npm test'
padd lint 'npm run lint'
p test
```
